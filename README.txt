# محلات الحاج أبو ياسر زعرب وأولاده – أبو عماد

## تشغيل البرنامج (على iPhone)
1) فك الضغط عن الملف.
2) ارفع المجلد على GitHub Pages (أو افتحه محلياً من خلال أي استضافة بسيطة).
3) افتح الرابط من Safari.
4) اضغط زر المشاركة ثم: **Add to Home Screen** (إضافة إلى الشاشة الرئيسية).

## مميزات
- إدارة الأصناف (إضافة/تعديل/حذف)
- شراء (يزيد المخزون) + سجل مشتريات
- بيع (يخصم المخزون) + إنشاء فاتورة تلقائياً برقم تسلسلي
- طباعة الفاتورة
- تقارير يومية (شراء/بيع) + طباعة تقرير
- تنبيه مخزون منخفض
- تصدير/استيراد نسخة احتياطية JSON

> كل البيانات محفوظة داخل الجهاز فقط (LocalStorage).
